1. В середені батьківського елемента якщо хочемо посунути дочірні елементи це слід робити by paddings
Тоюто дочірні елементи в середені батьківського посуваються by paddings
2. Через #ID в CSS не рекомендовано звертатись. Краще через класи
ID використовуються, зокрема, для посилань для якорів (anchor)
3. Section недоречно посувати by margin. Слід використовувати paddings.
Використовувати pattern:
    .section {
        padding: 48px 0;
    }
4. Сховати елемент можна за доаомогою style display: none
5. Коли задається width or hight in px - ви щось робите не так.
Ширина і висота блоку повинна залежати від контенту в середені нього.
Для цього задаємо:
    .container {
    width: 100%;
    max-width: 1240px; 
    ...
